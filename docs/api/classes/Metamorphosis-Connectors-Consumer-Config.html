Config extends AbstractConfig
in package
  • Application

Config.php : 18

This class is responsible for handling all configuration made on the kafka config file as well as the override config passed as argument on kafka:consume command.

It will generate a runtime configuration that will be used in all classes. The config will be on Manager singleton class.
